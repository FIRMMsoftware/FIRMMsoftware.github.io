Framewise Integrated Real-time MRI Monitoring

Easy to set up and use, the FIRMM software suite provides MRI scanner operators with data quality metrics in real time. Using FIRMM to identify the ideal scan time for each person can reduce total brain MRI scan times and associated costs by 50% or more.

Find Out More Download

The problem

Some subjects make it easy to collect high quality data...




... some don't...




... and with most, it's hard to tell just by looking.

The FIRMM solution

Our software suite allows you to measure data quality in real time, easily, intuitively, and accurately. Here's what you can do with FIRMM.

View framewise displacement calculations (i.e. movement) in real time.

Based on user input standards, see how much quality data is collected at a glance.

Predict how much scan time it will take to collect all the quality data you need.

Stop scanning when your goals are met.

FIRMM is availiable for download on the Neuroimaging Informatics Tools and Resources Clearinghouse (NITRC). Documentation is hosted at Read the Docs.
